# NSI_Pascal

Ce projet s’intitule « Projet_decodeur ». Il a été réalisé par Noam, Prasaad, Malek et Gabriel, sous la direction de Madame Grancher. La date de réalisation est décembre 2025-février 2026 et la version actuelle du programme est la version 1.0. Le programme a été développé en Python 3 et fonctionne avec Python 3.8 et les versions ultérieures. Le fichier principal à exécuter pour lancer le programme est main.py. Le projet est composé de plusieurs fichiers : un fichier principal qui gère le menu et l’exécution du programme, ainsi que des fichiers contenant les fonctions de conversion. Le programme permet à l’utilisateur de choisir la base d’un nombre (décimal, binaire, octal ou hexadécimal), de saisir un nombre, puis d’obtenir différentes représentations : en décimal, en binaire, en hexadécimal et en complément à 2 sur 8 bits. Les paramètres utilisés concernent notamment le nombre à convertir et le nombre de bits pour le calcul du complément à 2. Le programme utilise uniquement des fonctions natives de Python comme int(), abs(), input(), print(), upper() et isdigit(), et n’utilise aucun module externe. L’arborescence du projet est simple, avec un dossier principal contenant le fichier principal, les modules de conversion et le fichier README. Ce projet a pour objectif de mieux comprendre les bases numériques et les conversions entre différents systèmes de numération.
