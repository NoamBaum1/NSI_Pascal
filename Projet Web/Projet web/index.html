Expérience immersive

Dans cette expérience immersive :

- Vous incarnerez un jeune étudiant, vous venez de sortir du lycée et vous ne savez pas vraiment quoi faire de votre vie. Vous scroller sur Tiktok et un jour vous entendez parler de jeune qui crée des sites web pour des entreprises. Tout ce qui vous a vraiment intéressé c’est l’argent que vous pouvez gagner. Donc vous commencer à effectuez vos recherches et vous tombez sur une formation pour apprendre l’histoire du web. Sans hésitez,vous sortez votre carte bleue et payez 4000 euros pour cette soi-disant formation miraculeuse. Puis le lendemain, vous recevez un mail avec un lien pour la formation...
